📝 Text Analyzer System

📌 Project Description

This is a beginner Java mini project based on String concepts.

The project analyzes text and performs different string operations using Java.

---

🚀 Features

- Count vowels
- Count consonants
- Reverse text
- Palindrome check
- Character frequency count
- Compare strings
- Convert to uppercase
- Convert to lowercase

---

🛠 Technologies Used

- Java
- Strings
- Loops
- Conditional Statements
- Arrays

---

▶ How to Run This Project

Compile the program

javac TextAnalyzerSystem.java

Run the program

java TextAnalyzerSystem

---

📷 Example Output

Enter the String:
Madam

Original Text is :Madam

vowel count : 2
consonant count : 3

reverse String is madaM

String is Palindrome

Frequency of Char is: m = 2
Frequency of Char is: a = 2
Frequency of Char is: d = 1
