🎯 Pattern Generator Mini Project (Java)
This is a simple Java console project that I built while learning programming. It generates different star and number patterns using loops.
I created this project to practice Java basics and improve my logic-building skills.

📌 About This Project
In this project, I tried different pattern programs using loops in Java. While working on it, I understood how nested loops work and how spacing changes the output.
This project helped me think step by step and improved my problem-solving skills.

🧠 What I Learned
Java basics
How loops work (for loop and nested loop)
How to build logic for patterns
Importance of spacing and alignment
Step-by-step problem solving

⭐ Patterns I Practiced
Pyramid pattern
Reverse pyramid pattern
Right-aligned triangle
Number patterns

💻 How to Run This Project
Open the project in VS Code or any Java IDE
Compile the file:

javac PatternGenrator.java

Run the program:

java PatternGenrator
