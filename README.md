🎨 Pattern Generator Mini Project

📌 Project Description

This is a beginner Java mini project based on loops and pattern printing.

The project generates different star patterns using nested loops in Java.

---

🚀 Features

- Right Triangle Pattern
- Center Pyramid Pattern
- Inverted Pyramid Pattern
- Reverse Triangle Pattern
- Loop Practice Programs

---

🛠 Technologies Used

- Java
- Nested Loops
- Conditional Statements

---

▶ How to Run This Project

Compile the program

javac PatternGenerator.java

Run the program

java PatternGenerator
