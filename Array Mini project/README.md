Student Result Management System 📊

📌 Project Description

This is a beginner Java mini project based on Arrays.

The project manages student marks and performs multiple array operations using Java.

---

🚀 Features

- Input student marks
- Display student marks
- Calculate total marks
- Calculate average marks
- Find highest marks
- Find lowest marks
- Count even and odd marks
- Linear search for marks
- Bubble sort student marks
- Reverse sorted array

---

🛠 Technologies Used

- Java
- Arrays
- Loops
- Conditional Statements
- Bubble Sort
- Linear Search

---

📚 Concepts Covered

- Array Traversal
- Searching
- Sorting
- Min/Max Logic
- Sum and Average
- Reverse Array Logic

---

▶ Sample Output

Enter number of students: 5

Enter student marks:
45 78 90 67 88

Highest Marks = 90
Lowest Marks = 45
Average Marks = 73.6

Sorted Marks:
45 67 78 88 90
